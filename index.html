Organic Course Visualisation & Recommendation [Orgarec]

This is an EPFL semester project exploring what kind of visualisations could be suitable to explore study plans, based on which students took what course.

It is Scala/Lift based. Depending on your interest, you might want to only consider one of the two following parts of the project:

View

The view sub-project holds everything related to visualization: if you like plotting and drawing stuff, start here.

However, if you want to plot the data of some very specific queries, you'll have to dive into the core and write your own cute stored procedure(s).

Core

The core holds the storage, model and computation layer.

If you're only interested in raw data, want to represent study-plans or course histories for students, forget about the view and its nasty JavaScript and go directly for the core.

Documentation

The automatically generated scaladoc can be browsed for the core and the view.

You may also generate it yourself by checking out the project and running sbt doc to be sure to have the latest (hopefully helpful) comments.

And of course, if you have any unresolved question, try an e-mail ! (I can't guarantee a timely reply if we are currently running around the globe...)
